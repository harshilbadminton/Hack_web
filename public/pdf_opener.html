Pdf Opener Tutorial-By Harshil Jain

Here in this pdf opener we are using both front-end and backend javascript (i.e node.js) with express to create a website that openes pdf

Configure the server for node.js and install express for backend server and nodemon for starting the server in which the server will automatically get refreshed once the changes are committed

Create the basic html layout 

                
                  <div id="pdf-container" class="bg-white p-4 shadow-md rounded-lg"></div>
              
                  <form id="upload-form" enctype="multipart/form-data">
                      <input type="file" name="pdf-file" id="pdf-file-input">
                      <button type="submit" class="bg-black text-white font-bold text-1xl text-center border-[0.5vw] border-orange-500 w-[10vw] h-[8vh] hover:bg-orange-300 hover:text-black">Upload</button>
                  </form>
                  <iframe id="pdfFrame" width="100%" height="600px"</iframe>

Give styling to the iframe where the pdf will get displayed 

                
                    iframe {
                        border: 1px solid #ccc;
                        background-color: #f5f5f5;
                        background-color: black;
                        /* Add more CSS properties as needed */
                      }
