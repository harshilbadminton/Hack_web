Scanning IP Addresses and Ports with Python

Welcome to the Hacker's Den! In this blog post, we'll explore how to use Python to scan IP addresses and ports for potential vulnerabilities like a pro coder.

To get started, we need to import the necessary libraries:

import socket
import asyncio

Next, let's prompt the user for the IP addresses and the number of ports they want to scan: 

targets = input("[+] Enter the IP addresses you want to scan (separated by ','): ")
ports = int(input("[+] Enter how many ports you want to scan: "))

Now, let's define some asynchronous functions to perform the scanning:

async def splitter():
    if ',' in targets:
        target = targets.split(", ")
        for word in target:
            await scan(word, ports)
    else:
        await scan(targets, ports)


async def scan(word, ports):
    for port in range(1, ports):
        await scan_port(word, port)


async def scan_port(word, port):
    sock = socket.socket()
    try:
        sock.connect((word, port))
        print("[*] Port opened: " + str(port))
    except:
        pass
    finally:
        sock.close()

Finally, it's time to unleash your hacker skills! Run the code and observe the magic:

asyncio.run(splitter())
